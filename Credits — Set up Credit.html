Tips for Setting Up Credits Using a File
Avoiding Upload Errors
Keep the file format as provided — avoid adding, removing, or rearranging columns to ensure successful processing.
Client Name Matching
Client names in the file must match exactly as they appear in Sport XT. Use the reference list on the Stakeholders page if unsure.
Credit Amounts
Enter amounts as whole numbers without currency symbols or commas. The system will apply the tournament's default currency (AUD).
Overwriting Existing Credits
If a client already has a credit, uploading a new file will overwrite it after review. Previous credit history is always preserved in the audit log.
